Detailed Description


This class describes a color property composed of 4 GLfloat parameters, R,G,B and A respectively.

Colors can be defined by settings each component individually or by using the predefined colors from the palette. These colors can be initialized by calling the different public methods of the class.

For gray scales, you may use method setGrayLevel() that allows the programmer to finely adjust the gray scale level from 0.0 - 1.0.
