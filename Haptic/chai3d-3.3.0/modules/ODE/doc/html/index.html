Introduction

This module is an extension to the core CHAI3D framework that incorporates support of the Open Dynamics Engine (ODE) developed by Russel Smith.
For a general introduction to the ODE libraries, we recommend to review the documentation and examples that are available on the official ODE website.
See www.ode.org for more information about ODE.

See www.chai3d.org for more information about CHAI3D.

IMPORTANT NOTE: the online documentation for the CHAI3D libraries is a continuous work-in-progress.
Although the large majority of classes have been documented properly, there might still be some poorly documented items. If you happen upon an undocumented or poorly documented class and / or class method which you find hard to understand, please give us a notice so we can rectify the situation.
